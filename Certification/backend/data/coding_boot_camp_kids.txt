Coding Boot Camp for Kids
The Coding Boot Camp for Kids is organized by Code Warriors Academy. This program falls under the Community category. It is designed for participants in the Ages 8-16 age range.

Program Features:
This program offers several valuable features including Small class sizes (max 10 students), Project-based learning, Take-home computers for practice, Certificate of completion, Portfolio of projects, and Free coding resources.
